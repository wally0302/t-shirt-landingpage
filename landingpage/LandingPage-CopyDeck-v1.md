## 2. Problem → Solution
### 問題
- 新買的衣服拍完照，過幾天就找不到
- 相簿很滿，但穿搭資訊很難整理
- 出門前要在衣櫃與天氣 App 之間來回切換

### 解法
- 上傳後立即分類，建立可查找的衣櫃
- 用分類、顏色、備註快速篩選
- 同頁查看天氣，減少切換成本

## 3. Feature Cards
### 卡片 1：快速上傳
- Title：30 秒新增一件衣物
- Body：支援 JPG / PNG / WEBP，上傳後自動處理顯示尺寸。

### 卡片 2：快速找到
- Title：搜尋、分類、排序一次搞定
- Body：不再靠相簿硬找，出門前直接定位你要的單品。

### 卡片 3：天氣整合
- Title：穿搭決策更直覺
- Body：可依定位或城市查看溫度、體感、降雨與風速。

### 卡片 4：整理節奏
- Title：每日整理習慣
- Body：透過每日上傳額度，讓衣櫃管理維持穩定節奏。

## 4. How It Works
- Step 1：登入帳號
- Step 2：上傳衣物並加上分類/備註
- Step 3：出門前依天氣與搜尋快速找到要穿的衣服
